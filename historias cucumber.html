US01 Gestionar información de monitores
Feature: Gestionar información de monitores
     Para poder gestionar información de monitores
     Un gerente
     Tiene que poder añadir, listar, modificar y borrar los monitores del sistema

     Scenario: Dar de alta un nuevo monitor
          Given Yo estoy logueado
          When Yo doy de alta un nuevo monitor
          Then Yo debería ver al nuevo monitor en el sistema.

     Scenario: Listar monitores
          Given Yo estoy logueado
          When Yo voy al apartado de monitores en el sistema
          Then Yo debería ver la lista de los monitores dados de alta.

     Scenario: Modificar un monitor
          Given Yo estoy logueado
          When Yo modifico los datos de un monitor
          Then Yo visualizo los datos del monitor cambiados

     Scenario: Dar de baja un monitor
          Given Yo estoy logueado
          When Yo doy de baja a un monitor
          Then Yo no debería ver al monitor en el sistema
